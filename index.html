[01] — About

A short story

Two minds.
One craft.

I'm Abhinav — a QA Engineer who started as a Java developer and never stopped switching hats. I build with Spring Boot, ship reliable APIs, and design Figma interfaces. Then I put on the other hat and test the hell out of them with Playwright, Selenium and a healthy paranoia.

At Mozilor, I write Playwright suites that test plugins across WordPress, Shopify and Wix. At UST Global, I shipped functional and regression coverage that the team relied on. Before all that, I built REST APIs at Maverix Pro.

I think the best testers know how code feels from the inside. I think the best developers know exactly how it can break. I want to be both.
